About My Portfolio


This website is to serve the purpose of showing off my knowledge on certain concepts. In order to suppliment my resume, this website will have actual code I have written and assignments that I have done while at the University of Louisville. Instead of showing you a screen shot of one code sample, I put my code on Github which will allow you to view the entirety of the file.

On the navbar at the top, there is a "Sections" tab, which will have all work I have done broken up into specific areas. For example, there is a development section where I put all work I have done that relates to writing code. Some of the links take you to Github, others will open a document/picture in an iframe on the page.

Also on the navbar is my name. When clicked, a dropdown menu displays different ways of contacting me. These same ways are located on the right of this page
