Logic

Another assignment was to create an application to display when a student can register for a class. At the University of Louisville, if every student can register for classes all at once, the servers would crash. In order to make it a smoother process, the university splits up the registration window depending on your year and last name. Seniors register first, then the Juniors, and so on. Every seemster the last name changes around though. One semester, a student with the first letter of their last name being 'S' can register from 8am-10am, whereas the next semester they will register between 2pm-4pm. This application has a person click their year as well as type in their last name and the output is the registration window.

There are two versions of the application. The first one is using a bunch of if-else statements, and the second application is cleaner, using arrays.

Library Application

For the entire semester of Software Development 2, we were working on creating a large application. We had multiple iterations of the assignment, leading up to a fully functional application.

The goal of the application was to make a library. The user can input new books, CD's, DVD's, Blu-Ray's, etc. into the system. The user can then save it and all changes will be saved to a file. When opening the application again, the user can go to File > Open then browse the computer for the save file. The user can also search for items in the library and check them in or check them out.
